## 💎 Elevate Your Career Narrative

**Vitae** is not just another resume builder; it’s an elite, AI-orchestrated platform designed for high-impact professionals. Built with a "Brutalist-Premium" aesthetic, Vitae combines cinematic UI with advanced LLM intelligence to transform your career history into a compelling narrative that beats the bots and wows recruiters.

### 🚀 Key Capabilities

- **🧠 Global AI Chat Assistant**: A persistent, intelligent companion that helps you brainstorm, rewrite, and optimize your resume from any page.
- **🎯 Job Match Engine**: Advanced ATS analysis. Upload your target job description and get real-time feedback on your compatibility, keyword gaps, and tailoring strategies.
- **✍️ Cover Letter Studio**: Draft personalized, high-conversion cover letters in seconds, tailored to specific company cultures and job roles.
- **📄 Premium Templates**: Access a curated collection of high-fidelity templates, including "London Luxury," "Consulting Elite," and "Harvard Classic."
- **⚡ One-Click Import**: Instantly convert your existing PDF or Word resume into our intelligent schema using AI-driven parsing.
- **🖨️ Pixel-Perfect PDF Export**: High-fidelity rendering using serverless Puppeteer to ensure your resume looks identical on every screen and printer.

## 🛠️ The Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router, Server Components)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) + [Base UI](https://base-ui.com/)
- **State Management**: [Zustand](https://zustand-demo.pmnd.rs/)
- **Intelligence**: [Groq AI](https://groq.com/) (Llama 3.1 70B)
- **Database**: [Prisma](https://www.prisma.io/) + [PostgreSQL](https://www.postgresql.org/)
- **Animations**: [GSAP](https://greensock.com/gsap/) + [Framer Motion](https://www.framer.com/motion/)

## ⚙️ Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/ResumeBuilder.git
   cd vitae
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Configure Environment Variables**
   Create a `.env.local` file:
   ```env
   DATABASE_URL="your-postgresql-url"
   GROQ_API_KEY="your-groq-key"
   NEXT_PUBLIC_APP_URL="http://localhost:3000"
   ```

4. **Launch Development Server**
   ```bash
   npm run dev
   ```
